# Repositorio-1004


1. ¿Qué diferencia encontraste entre trabajar en computador y celular?
En el computador es más cómodo y rápido trabajar porque hay mejor visualización y más herramientas disponibles. En cambio, en el celular es más limitado y puede ser más difícil escribir y navegar, aunque sirve para tareas básicas.
2. ¿Qué ventajas tiene GitHub para guardar proyectos?
GitHub permite guardar proyectos de forma segura en la nube, tener un historial de cambios, trabajar en equipo y acceder a los archivos desde cualquier lugar.
3. ¿Por qué es importante aprender herramientas digitales colaborativas?
Porque ayudan a trabajar mejor en equipo, organizar proyectos y son muy importantes en el mundo laboral actual.
4. ¿Qué fue lo más difícil del proceso?
Lo más difícil fue entender cómo hacer commits correctamente y usar algunas funciones desde el celular.
5. ¿Cómo imaginas usar GitHub en el futuro?
Lo imagino como una herramienta para guardar mis proyectos, colaborar con otras personas y mostrar mis habilidades en programación.
